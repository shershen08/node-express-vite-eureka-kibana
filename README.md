
# Express

App using Node+ExpressJS to read data from MongoDB

Frontend with Vue3 + Vite

Microserice setup:

- logs via Kibana
- overview with Eureka
- controlled with pm2


## Setup

- mongo connected nodejs: 3001
- sample express app: 3000
- fronend: 3002
